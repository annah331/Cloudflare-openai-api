# Cloudflare OpenAI API Integration (Frontend Demo)

## 📌 Overview
A frontend interface designed to connect to OpenAI through Cloudflare Workers.

The project demonstrates API integration structure and secure backend routing concepts.  
Currently deployed as a frontend demo due to API cost limitations.
